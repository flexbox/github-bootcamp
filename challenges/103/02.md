- [ ] Create a **Schedule Event**

```yml
name: Nightly build
on:
  schedule:
    - cron:  '0 0 * * *'

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout code
        uses: actions/checkout@v2
      - name: Run a one-line script
        run: echo Hello, Scheduled Event!        
```

In this example, the workflow runs every day at midnight (UTC).
